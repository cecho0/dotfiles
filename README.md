### install
``` bash
# install python dep on root
python3 -m pip install requests
python3 -m pip install tqdm
python3 -m pip install pyyaml

# install python dep on simple user
pip3 install requests
pip3 install tqdm
pip3 install pyyaml

# download, install software and configuration default, some options will be control in config.yaml
python3 setup.py
```
